# Security Policy

This project is a local simulation tool and does not collect or process user data. We actively maintain and support the latest version of this project. If you are using an unsupported version, please upgrade to the latest release before reporting vulnerabilities.

We still welcome reports of potential security issues such as:
- Unsafe file handling
- Dependency vulnerabilities
- Code execution or sandboxing issues

If you find a potential issue, please email **samuelprado730@gmail.com**. Thank you for helping us keep the project secure.
